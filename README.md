# Ecommerce Agave

Plataforma e-commerce desarrollada en **Laravel** con **Bootstrap**, enfocada en la venta de bebidas espirituosas de maguey y agave.  

## 🚀 Módulos principales
- Usuarios (registro, login, roles)
- Productos (catálogo, categorías, marcas, etiquetas)
- Carrito de compras
- Checkout y pedidos
- Envíos
- Cupones y descuentos
- Atributos dinámicos de productos

## 🔧 Tecnologías
- Laravel 10+
- MySQL
- Bootstrap 5
- GitHub (colaboración en equipo)
